# Russian Diceware 4d6

## Структура репозитория

Репозиторий разделён на две верхнеуровневые зоны:

- `dictionary/` содержит конечные пользовательские артефакты Russian Diceware 4d6.
- `source/` содержит исходные данные и материалы сборки, из которых получаются эти конечные артефакты.

```text
/
├── dictionary/
│   ├── editions/
│   ├── wordlists/
│   └── docs/
└── source/
    ├── data/
    │   ├── external/
    │   ├── roots/
    │   └── ...
    └── pipelines/
        ├── roots/
        └── ...
```

## Что где лежит

- `dictionary/editions/` содержит редакции словаря, предназначенные для чтения с экрана и/или печати.
- `dictionary/wordlists/` содержит списки слов в формате TSV для программного использования.
- `dictionary/docs/` содержит пользовательскую документацию по использованию словаря.
- `source/data/` содержит внешние входные данные, промежуточные артефакты и данные отдельных процессов сборки.
- `source/pipelines/` содержит артефакты, управляющие сборкой: скрипты, промпты, конфиги и служебные пояснения.
